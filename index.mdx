## Get started

<Steps>
  <Step title="Create a new project">
    Scaffold a fresh Nitro project using the official starter:

    ```bash
    npx create-nitro-app
    ```
  </Step>
  <Step title="Add to an existing Vite app">
    Install Nitro and add the Vite plugin to your config:

    ```bash
    npm install nitro vite
    ```

    ```typescript vite.config.ts
    import { defineConfig } from "vite";
    import { nitro } from "nitro/vite";

    export default defineConfig({
      plugins: [nitro()],
    });
    ```
  </Step>
  <Step title="Create your first route">
    Add a file to `routes/` — the filename maps directly to the URL path:

    ```typescript routes/api/hello.ts
    import { defineHandler } from "nitro";

    export default defineHandler((event) => {
      return { message: "Hello from Nitro!" };
    });
    ```
  </Step>
  <Step title="Start the dev server">
    ```bash
    npm run dev
    ```

    Your API is now live at `http://localhost:3000/api/hello`.
  </Step>
</Steps>
